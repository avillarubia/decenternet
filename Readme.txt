# REACT PROJECT SETUP GUIDE
1. Use cmd (much better with admin rights)
2. Type: npx create-react-app project-name

# REACT REDUX 
1. In the browser download the Redux DevTools extension
2. To check the state managed by redux, while the app is running, right-click the browser and click Redux Devtools

# TO USE THE APP
1. Register https://decenternet-app.herokuapp.com/register
2. Then login https://decenternet-app.herokuapp.com/login
3. Click add a book.
